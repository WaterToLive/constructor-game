# constructor-game 
## Shapes are falling from the top of the canvas
- create canvas (full screen)
- create 3 files Square.js, Circle.js and Triangle.js
- use a constructor function to draw each shape on the canvas
- move the shapes
  
# Tap the shapes to get points
- Different shapes mean different amount of points. 
# The more points you get the faster the shapes come down. 
- IF/WHEN a shape hits the bottom of the canvas Promt "GAME OVER" 
# Restart the game.
